# Titel/Honours-Hexfelder Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 12 „Honour"-Hexfelder (CL, Weltmeister, 5 Meistertitel, 5 Pokalsiege) ins Spiel bringen, erfüllt, wenn der Spieler den Titel gewonnen hat.

**Architecture:** Neuer Hexfeld-Typ `honour`, geprüft gegen ein neues, in der Pipeline berechnetes Spielerfeld `t:[...]`. Spiel-Seite (Logik + Rendering) ist lokal/TDD; die Honours-Daten entstehen in der Daten-Pipeline (Kaggle) aus Meister-Berechnung (Punkte), Finalspielen (+ Override für Elfmeter-Finals) und kuratierten WM-Kadern.

**Tech Stack:** Vite + React 18 (ESM). Tests: Node `node:test`. Pipeline: Python/pandas auf Kaggle (Dataset `davidcariboo/player-scores`).

**Phasen:**
- **Phase 1 (Tasks 1–3): Spiel-Seite** — lokal, TDD, subagent-tauglich. Nutzt synthetische `t`-Daten in Tests.
- **Phase 2 (Tasks 4–8): Daten-Pipeline** — Controller + Nutzer (Kaggle-Läufe, kuratierte Listen), Verifikation über Prüfberichte. Wird nach einem Probe-Lauf finalisiert.

Alles läuft auf Branch `feat/honours`; Merge erst, wenn Spiel-Seite **und** Daten fertig sind (sonst unlösbare Honour-Felder live).

```js
export function buildBoardSerial() {
  const nLeague = 1 + Math.floor(Math.random() * 3); // 1–3
  const nHonour = 2 + Math.floor(Math.random() * 3); // 2–4
  const leagues = pick(LEAGUES, nLeague);
  const honours = pick(HONOURS, nHonour);
  const specials = pick(SPECIALS, 3);
  const blClubs = pick(CLUBS.filter((c) => c.lg === "BL"), 4);
  const nations = pick(NATIONS, 6);
  const rest = pick(CLUBS.filter((c) => !blClubs.includes(c)), 31 - 3 - 4 - 6 - nLeague - nHonour);
  const chosen = shuffle([...specials, ...blClubs, ...nations, ...rest, ...leagues, ...honours]);
  return chosen.map((d) => ({ t: d.type, k: d.key }));
}
```

Invariante: `3 + 4 + 6 + (16 − nLeague − nHonour) + nLeague + nHonour = 31` (rest min = 16−3−4 = 9).

- [ ] **Step 1: Berechnungs-Funktionen in `honours.py` ergänzen**

```python
import pandas as pd, unicodedata

def _norm(s):
    s = unicodedata.normalize("NFD", str(s)).encode("ascii", "ignore").decode()
    return s.lower().strip()

def league_champions(games, comp_id, since=2000):
    """ -> {season: champion_club_id} aus Punktetabelle."""
    g = games[(games["competition_id"] == comp_id) & (games["season"] >= since)]
    g = g.dropna(subset=["home_club_goals", "away_club_goals"])
    out = {}
    for season, gs in g.groupby("season"):
        pts, gd, gf = {}, {}, {}
        for _, m in gs.iterrows():
            h, a = m["home_club_id"], m["away_club_id"]
            hg, ag = int(m["home_club_goals"]), int(m["away_club_goals"])
            for c in (h, a):
                pts.setdefault(c, 0); gd.setdefault(c, 0); gf.setdefault(c, 0)
            gd[h] += hg - ag; gd[a] += ag - hg; gf[h] += hg; gf[a] += ag
            if hg > ag: pts[h] += 3
            elif ag > hg: pts[a] += 3
            else: pts[h] += 1; pts[a] += 1
        if pts:
            out[season] = max(pts, key=lambda c: (pts[c], gd[c], gf[c]))
    return out

def cup_winners(games, comp_id, penalty_finals, honour_key, since=2000):
    """ -> ({season: winner_club_id}, [unresolved (key,season)]) aus Finalspiel."""
    g = games[(games["competition_id"] == comp_id) & (games["season"] >= since)]
    fin = g[g["round"].astype(str).str.contains("Final", case=False, na=False)]
    out, unresolved = {}, []
    for season, fs in fin.groupby("season"):
        m = fs.iloc[-1]  # letztes Finalspiel der Saison
        hg, ag = m["home_club_goals"], m["away_club_goals"]
        if pd.notna(hg) and pd.notna(ag) and int(hg) != int(ag):
            out[season] = m["home_club_id"] if int(hg) > int(ag) else m["away_club_id"]
        elif (honour_key, season) in penalty_finals:
            out[season] = penalty_finals[(honour_key, season)]
        else:
            unresolved.append((honour_key, season))
    return out, unresolved

def squad(appearances, comp_id, club_id, season):
    """player_ids mit >=1 Einsatz in comp_id für club_id in season."""
    a = appearances
    return set(a[(a["competition_id"] == comp_id) & (a["player_club_id"] == club_id)
                 & (a["season"] == season) if "season" in a.columns else
                 (a["competition_id"] == comp_id) & (a["player_club_id"] == club_id)]["player_id"])
```
